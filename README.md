# Lou's Team PWA

Next.js 14 App Router Projekt für Lou’s Foodtruck Team-PWA mit drei Kernfunktionen:

- **Schichten-Board**: Übersicht aller Einsätze mit Mail- und WhatsApp-Deep-Links.
- **Reparatur-Meldung**: Formular inkl. optionalem Datei-Upload.
- **Feedbackbox**: Anonyme/sensible Rückmeldungen mit optionaler Antwortmöglichkeit.

## Entwicklung

```bash
npm install
npm run dev
```

Alle Funktionen laufen aktuell gegen Stubs und können später mit der FileMaker Data API und echter Authentifizierung verbunden werden.
